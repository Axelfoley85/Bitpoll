{% trans 'The History of BitPoll' %}

{% blocktrans %}BitPoll is a free software alternative to the Doodle website. Its origins are in Dudle from TU-Dresden. Later Dudel was written from Paul Bienkowski to speed it up and add a nicer Interface and some new Features. This is writen in Flask a Python web Framework{% endblocktrans %}

{% blocktrans %}After some time in Production use the discovered some shortcomings, especially with ldap and very big polls in the architecture. We decided to rewrite it again this time using Django, also in Python. For the starting point most styles and javascript artifacts where imported from the Dudel project.{% endblocktrans %}

{% blocktrans %}BitPoll was developed by Students from the Informatics department at University Hamburg. The Primary goal was to use it as one of the services provides on mafiasi.de as a Service for other Students. After a time to long, it is now ready for Productive use. The sourcecode can be found at GitHub. {% endblocktrans %}

{% blocktrans %}There are multiple Ideas for new Features to improve BitPoll further on. Hopefully we will find the Time to work on some of them{% endblocktrans %}
